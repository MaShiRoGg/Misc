##  2. <a name='-1'></a>基本语法

###  2.1. <a name='-1'></a>赋值

####  2.1.1. <a name='-1'></a>语法
```py
i = 1
i = 5 + 5
s = "这是个字符串"
b = True
```

####  2.1.2. <a name='-1'></a>解释
i = 1 代表将右值数字 1 赋值给变量 i，此时 i 的的值就是num 1  
换句话就是 i 被赋予了一个值num 1，简称 i 被赋值为 1  

i = 5 + 5 代表 i 被 重新赋予了一个值 5 + 5，也就是10，此时 i 为 10，  
原本的 1 被替换为了 10  

s = "这是个字符串" 表示 s 被赋值为 "这是个字符串"，此时 s 为 "这是个字符串"  

> 注意：命名时不能以数字和特殊字符开头，只能用字母和下划线_  
> 命名时最好使用小驼峰命名法，单词开头小写后续单词开头大写 例 thisIsExample

###  2.2. <a name='-1'></a>常用的运算符

常用的基本数学运算符有 + - * / ** // %

####  2.2.1. <a name='-1'></a>计算运算符
\+ \- \* / 对应为加减乘除  

\*\* 获取乘方 例如 i = 2 \*\* 3 此时 i 为 8  

// 获取整数商，也就是不留小数  
例如 i = 7 // 2 此时 i 为 3  

% 获取余数，例如 i = 7 % 2 此时 i 为 1  


> 计算中的优先级都是按照现实运算优先级，  
> 不确定可以加()来强制进行优先运算  
> \+ 运算符在 字符串中的使用:  
> `welcome = "hello" + "minecraft"`  
> 此时 welcome 的值为 "hellominecraft"  
> `tmp = "hello" + 1`  
> 此时 tmp 的值为 "hello1"

####  2.2.2. <a name='-1'></a>比较运算符
==  !=  <  <=  >  >=  

***==***   
比较左右两边是否相等，相等时返回 True
```python
b = (1 == 3 - 2)   # 此时 b 的 值为 True
b = (1 == 2)       # 此时 b 的 值为 False
print(b) # 输出 False
```
> 返回    
> 返回值就是得到的结果，基本类型的返回值就是它本身
> 比如 i = 1， 1 的返回值就是它本身 1  
> 所以 i 被 赋值 为 1 的返回值 1  
> i = 2 + 2
> 返回值就是2 + 2运算的结果 4  

> 注释  
> 在python中可以用#来表示单行注释  
> 释掉的内容不会被程序执行  
> 多行注释可以用三引号 """ 表示
> ```python
> # 单行注释
> print("hello world")
> """
> 这里是
> 多行注释
> """
> ```
> 注释可以用于记录一些方法或者对象的作用等


***!=***  
与 == 相反

***<***   
比较 左值 是否小于 右值，小于时返回 True  

***\>***   
与 < 相反  

***<=***   
比较 左值 是否小于等于 右值， 小于等于时返  回 True

***>=***  
与 <= 相反

###  2.3. <a name='-1'></a>函数
如果安装好了python的环境可以编辑试试看   
代码：  
`print("hello world !")`

输出：  
`hello world !`

解释：
print是函数的名字，这是python自带的函数，
作用是打印出括号中的字符串参数的内容  
紧跟在print后面的括号内是要传入print函数的参数，  
参数可以有零个到多个，用英文逗号隔开，  
最终打印出输出内容 `"hello world !"`

####  2.3.1. <a name='-1'></a>自定义函数
代码：
```python
def myFunc():
    print("hello minecraft !")

myFunc()
myFunc()
myFunc()
```
输出：
```
hello minecraft !
hello minecraft !
hello minecraft !
```
解释：  
***def***   
表明这里要定义一个函数  *(define 定义)*

***myFunc***  
自定义函数的名称 *(my function 我的函数)*  
根据自己的命名习惯来命名  
函数和变量命名都推荐使用 [小驼峰命名法](https://www.cnblogs.com/zzzlight/p/13880922.html) 

***()***  
内部为空 表示不需要参数，  
  
***:***   
表示从符号开始下面空四格或一个tab空格的代码是函数的内部代码  
`print("hello minecraft !")` 的解释如同上一解释  
`myFunc()`表示这里调用了 自定义的函数 `myFunc`,  
调用了三次则打印了三次 `hello minecraft ! ` 

####  2.3.2. <a name='-1'></a>定义带有参数的函数
代码：
```python
def add(num1, num2):
    print(num1 + num2)

add(1, 2)
add(3, 9)
```
输出：
```
3
12
```
解释：  
***num1*** 和 ***num2***  
 表示这个函数调用时需要两个参数

***print***  
用来打印输出到控制台的函数

***add(1, 2)*** 和 ***add(3, 9)***  
表示调用了两次add函数  
分别给了参数 ***1, 2*** 和 **3, 9**  
最后分别输出了它们的和


####  2.3.3. <a name='-1'></a>定义带有返回值的函数
代码:
```python
def getSum(num1, num2):
    return num1 + num2
print(getSum(1, 1))
print(getSum(getSum(1, 2)), 3)
```
输出：
```
2
6
```
解释：
***return num1 + num2***    
 表示整个函数的返回值是 数字1 和 数字2 的和  
***print***   
打印出 ***getSum(1, 1)*** 的值  
***getSum(getSum(1, 2), 3)***   
这里将 ***getSum(1, 2)*** 的返回值
作为另一个 ***getSum*** 的第一个参数，也就是 3
最后总返回值就是 6
